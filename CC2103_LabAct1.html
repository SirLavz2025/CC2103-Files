CC 2103 - DATA STRUCTURES AND ALGORITHMS

Laboratory Activity 1: Data Structures & Abstract Data Types

CC2103 • Laboratory Activity using Apache Netbeans

Overview & Objectives

This laboratory activity introduces fundamental computer science concepts in Java, focusing on user-defined data types, the architectural differences between static and dynamic data structures, and the structural properties of linear versus non-linear organizations.

Step 1: Setting up Apache NetBeans

1. Create a New Project: Launch Apache NetBeans, go to File > New Project..., select Java with Ant > Java Application, and click Next.

2. Project Details: Name your project CC2103_Lab1Activity and select your preferred working directory. Click Finish.

3. Configure Package: Expand your project in the left navigation panel, right-click Source Packages, select New > Java Package..., and name it cc2103_lab1activity.

4. Create Source File: Right-click the new package, select New > Java Class..., and name the class CC2103_Lab1Activity.

Step 2: Source Code Implementation

Replace any boilerplate code in your newly created file with the complete implementation provided below:
